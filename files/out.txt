Tytuł: Dodanie książki
Aktorzy: [Bibliotekarz]
Aktor systemowy: System
1. Bibliotekarz wybiera opcje dodania nowej pozycji książkowej.
2. Bibliotekarz podaje dane książki.
3. IF: Bibliotekarz pragnie dodać egzemplarze książki.
   3.1. Bibliotekarz wybiera opcję definiowania egzemplarzy.
   3.2. System prezentuje zdefiniowane egzemplarze.
   3.3. FOR EACH egzemplarz:
      3.3.1. Bibliotekarz wybiera opcję dodania egzemplarza.
      3.3.2. System prosi o podanie danych egzemplarza.
      3.3.3. Bibliotekarz podaje dane egzemplarza i zatwierdza.
      3.3.4. System informuje o poprawnym dodaniu egzemplarza i prezentuje zaktualizowaną listę egzemplarzy.
4. nieBibliotekarz zatwierdza dodanie książki.
5. System informuje o poprawnym dodaniu książki.
